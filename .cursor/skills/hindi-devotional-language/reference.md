# Hindi devotional language — reference examples

## Katha

**Bad (literal):** पार्वती नंगी शिला पर सोईं।  
**Good:** पार्वती शिला या वन-भूमि पर शयन करती थीं।

**Bad:** कृष्ण नंगे पाँव दौड़े।  
**Good:** कृष्ण बिना जूते के दौड़कर मित्र को गले लगाया।

**Bad:** विनम्र भेंट तुच्छ थी।  
**Good:** विनम्र भेंट साधारण थी, पर हृदय से पूर्ण।

## Puja / verdict

**Bad:** यह व्रत महाशिवरात्रि से हल्का है।  
**Good:** यह मासिक अनुष्ठान महाशिवरात्रि की अपेक्षा संक्षिप्त है।

**Bad:** हवन के बाद व्रत हल्का करते हैं।  
**Good:** हवन के बाद व्रत समाप्त करते हैं।

## Tone check

Before shipping, ask:
- Would this word embarrass the reader if read to elders at puja?
- Does any line sound like the app is judging the user's devotion?
- Is any deity or devotee described with undressing or insult vocabulary?

If yes to any — rewrite.
