# Hindi worship glossary — Ganak canonical terms

**Owner-approved 2026-07-24.** Agents must use these spellings and senses in all festival/vrat Hindi copy. Do not invent synonyms for core ritual words.

## UI section labels (`VRAT_VIDHI_LABELS`)

| Key | Hindi (canonical) | Notes |
|-----|-----------------|-------|
| sankalpa | संकल्प | vow/intention at start of vrat |
| puja | पूजा | worship steps section |
| paran | पारण | breaking the fast — never पारणा |
| udyapan | उद्यापन | formal completion of a vowed series |
| vidhi | विधि | observance steps |
| stories | पढ़ने या सुनने योग्य क्षेत्रीय कथाएँ | regional kathas section |

## Core ritual vocabulary

| English | Hindi (canonical) | Avoid |
|---------|-----------------|-------|
| Sankalpa / vow | संकल्प | संकल्पना (abstract noun), resolution |
| Naivedya / food offering | नैवेद्य | केवल भोजन when meaning offering |
| Prasad | प्रसाद | after offering is received |
| Paran / break fast | पारण | व्रत खोलना (OK in prose), पारणा |
| Nirjala fast | निर्जला व्रत | निर्जल (short form OK in tight UI) |
| Udyapan | उद्यापन | समापन when meaning vowed completion |
| Aarti | आरती | आरती/आरतिक — use आरती |
| Arghya | अर्घ्य | अर्घ |
| Puja materials | पूजा-सामग्री | सामग्री alone in puja context |
| Family tradition | कुल-परम्परा | family custom calques |
| Temple tradition | मंदिर परम्परा | |
| With devotion | श्रद्धापूर्वक | भक्तिपूर्वक (OK), mechanically भक्ति से |
| Respectfully | आदरपूर्वक | |
| Fasting | उपवास or व्रत | context-dependent; major vrats → व्रत |

## Deity and address

| Use | Hindi |
|-----|-------|
| Lord Shiva (devotional) | भगवान शिव / महादेव |
| Goddess / Mata | माता, देवी |
| Greetings in copy | प्रभु, देवी, माता — not casual tu/tum for deities |
